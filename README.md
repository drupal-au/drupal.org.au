# Drupal.org.au
 
## Start the Drupalz

  * `composer install` - installs the prereqs
  * `lando start` - starts the containers
  * `ahoy sync` - syncs prod database to local
  * `lando drush -l https://drupal-org-au.lndo.site uli` - etc etc.
